# FLEET Webservice Kundenportal

Dieses Repository enthält die öffentliche Kundendokumentation und das aktuelle Installationspaket für den EHFleet Webservice.

Die Dokumentation wird mit MkDocs gebaut und über GitHub Pages veröffentlicht:

https://harzmann.github.io/ehfleet_webservice_docs/

Die Inhalte werden automatisch aus `harzmann/ehfleet_webservice` aktualisiert.
